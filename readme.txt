Url Tagging Builder help you tracking your marketing with custom campaign parameters Google Analytics.

== Description ==

This plugin help you add custom campaign parameters to your URLs. If you want to add Google Analytics to your WordPress blog, and then I advice you to see the <a href="http://wordpress.org/extend/plugins/google-analytics-for-wordpress/">Google Analytics for Wordpress Plugin</a>

== Installation ==
1. Make sure you have add Google Analytics to your WordPress blog. You can use <a href="http://wordpress.org/extend/plugins/google-analytics-for-wordpress/">Google Analytics for Wordpress Plugin</a>
2. Download and unzip plugin
3. Upload the 'url-tagging-builder' folder to the '/wp-content/plugins/' directory,
4. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Where can I find the result of my campaign performance? =

Open your analytics and click in the left hand side menu on Traffic Sources > Campaigns. Here you can see an overview of your marketing performance.

= where I have to put the custom campaign parameters URLs =

Use it on your email blast, tweet, your guest posting links, etc. Use it on your marketing channel
